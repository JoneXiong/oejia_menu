## oejia_menu
Odoo 菜单美化主题

## 特性
- 模块安装即可使用，轻量简单，不改变原有结构
- 支持 odoo 10、11 (12支持进行中)
- 支持 icon 图标，可以在odoo菜单页设置图片样式名 如“fa fa-circle-o
